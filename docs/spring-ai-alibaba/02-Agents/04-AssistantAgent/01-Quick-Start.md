# Assistant Agent 概览

Assistant Agent 是一个基于“代码即行动”（Code-as-Action）范式的企业级智能助手框架。

## 核心范式：Code-as-Action

与传统的选择工具（Tool Calling）模式不同，Assistant Agent 会根据用户需求**动态生成 Java/Python 代码**。
- **编排逻辑**: 它不是靠 LLM 决策下一步调用哪个 API，而是写一段逻辑来调用多个 API。
- **复杂任务**: 能够处理具有循环、判断、异常处理的复杂业务流程。

## 主要能力

- **环境安全执行**: 在沙箱或受控环境中执行生成的代码。
- **SPI 扩展点**: 开发者可以通过 SPI 轻松扩展助手的核心能力模块。
- **知识库接入**: 支持挂载外部知识库（RAG），使助手具备专业背景。

## 快速体验

1. 克隆项目并构建。
2. 提供具备代码生成能力的模型 API Key（推荐 Qwen-Max 或 GPT-4o）。
3. 使用内置的 Chat UI 界面开始交互。
4. 可以尝试发送如“帮我从文件夹中查找最近三天的销售日志，并计算平均客单价”等涉及工具编排的指令。
