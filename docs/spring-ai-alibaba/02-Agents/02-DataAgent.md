# DataAgent (NL2SQL)

DataAgent 是一个自然语言转 SQL (NL2SQL) 的智能体项目，允许用户使用自然语言直接查询数据库。

## 项目特色
- **免写 SQL**: 将自然语言需求转换为标准的 SQL 语句。
- **多数据库支持**: 基于 Spring AI 的抽象，可以适配 MySQL, PostgreSQL 等。
- **智能分析**: 不仅提供原始数据，还能根据查询结果给出业务建议。

## 工作原理
1. **Schema 获取**: 自动读取数据库表结构和备注信息（元数据）。
2. **Prompt 构造**: 将用户问题、表 Schema 和查询示例喂给 LLM。
3. **SQL 生成与校验**: LLM 生成 SQL，DataAgent 执行并验证正确性。
4. **结果呈现**: 将结果集转化为可视化表格或自然语言总结。

## 适用场景
- 企业内数据看板的快速提问。
- 非技术人员的数据自助服务。
- 复杂报表的初步生成。

## 快速启动
1. 准备业务数据库并配置数据源。
2. 配置 API Key。
3. 启动应用并打开提供的 Web 管理界面。
4. 在页面上点击“创建智能体”，配置对应的数据库实例后即可开始对话。
