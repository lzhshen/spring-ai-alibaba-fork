Spring AI Alibaba 是构建 Agent 智能体应用最简单的方式，只需不到 10 行代码就可以构建您的智能体应用。

![Architecture](https://java2ai.com/assets/images/architecture-new-5a8ace6f37598cf200e7c9bf379f14b4.png)

> **注意！** Spring Ai Alibaba 深度集成 Spring AI 生态，它是一个专为多智能体系统和工作流编排设计的项目。如果您正在寻找更高级的 ReactAgent 范式来构建模型驱动的智能体，请参阅 **[AgentScope](https://github.com/agentscope-ai/agentscope-java)** 项目。

Spring AI Alibaba 项目从架构上包含如下三层：

## Architecture Layers

*   **[[Notes/Agent Framework|Agent Framework]]**: 以 ReactAgent 设计理念为核心的 Agent 开发框架。
*   **[[Notes/Graph Runtime|Graph Runtime]]**: 低级别的工作流和多代理协调框架，Agent Framework 的底层运行时基座。
*   **[[Notes/Augmented LLM|Augmented LLM]]**: 为构建大型语言模型（LLM）应用提供基础抽象。

## Core Concepts

*   **[[Notes/ReactAgent|ReactAgent]]**: 构建具备推理和行动能力的智能代理。
*   **[[Notes/Multi-Agent Orchestration|Multi-Agent Orchestration]]**: 顺序、并行、路由、循环等多种多智能体协同模式。
*   **[[Notes/Context Engineering|Context Engineering]]**: 内置 Prompt 工程和上下文管理最佳实践。
*   **[[Notes/Human-in-the-Loop|Human-in-the-Loop]]**: 集成人工反馈和审批流程。
*   **[[Notes/Agent-to-Agent Communication|Agent-to-Agent Communication]]**: 基于 Nacos 的跨服务分布式 Agent 协作。

## Getting Started

*   **[[Notes/Spring AI Alibaba Installation|Installation]]**: Maven 依赖配置。
*   **[[Notes/Creating an Agent|Creating an Agent]]**: 快速创建一个天气助手 Agent。
